# ♻️ Smart Recycling AI System

## 📌 Overview
**Smart Recycling** is an AI-powered waste management system developed as my graduation project. This system integrates deep learning, Large Language Models (LLMs), and data analytics to optimize waste sorting, estimate carbon footprints, and provide personalized recycling insights. The goal is to reduce landfill waste and promote environmental sustainability through data-driven solutions.

## ✨ Key Features
* **Automated Waste Classification:** Utilizes a fine-tuned **ResNet50** (Deep Convolutional Neural Network) to accurately classify and sort different types of waste materials from images.
* **Intelligent Insights & Recommendations:** Integrates **DeepSeek R1** (Advanced LLM) to analyze consumer behavior and generate personalized, actionable advice on recycling habits.
* **Carbon Footprint Estimation:** Calculates and tracks emissions based on the sorted materials to encourage eco-friendly practices.

## 🛠️ Technologies & Tools Used
* **Programming Language:** Python
* **Computer Vision / Deep Learning:** ResNet50, PyTorch / TensorFlow
* **Large Language Models (LLMs):** DeepSeek R1
* **Data Processing:** Pandas, NumPy, Data Analytics
* **Environment:** Jupyter Notebook

## 🚀 How it Works
1. **Image Input:** The system receives an image of the waste item.
2. **Classification:** The ResNet50 model processes the image to identify the exact material type (e.g., plastic, metal, glass, paper).
3. **Data Analysis:** The system logs the item and calculates its associated carbon footprint.
4. **AI Recommendations:** DeepSeek R1 generates smart tips for the user on how to properly recycle the item and reduce future waste.
